== Description ==
<p>
This plugin is all-in-one slider solution for WordPress that create responsive(Premium Features) banners. 
</p>

Free Version features:
<p>
<ul>
<li>* Use shortcode to add it to any page at frontend.</li>
<li>* Click 'Add new banner' under 'Banner Ad Slideshow' menu at admin page to insert banner.</li>
</ul>
</p>

Premium Features:
<p>
<ul>
<li>* All of the above free version features. </li>
<li>* Click 'Banner Settings' under 'Banner Ad Slideshow' menu at admin page for configuration.</li>
<li>* Banner Ad Slideshow looks responsive on any device.</li>
<li>* The Banner Ad Slideshow description added.</li>
<li>* The Banner Ad Slideshow URL added.</li>
<li>* The Banner Ad Slideshow Expiring date added.</li>
<li>* Click 'Banner Manager' under 'Banner Ad Slideshow' to Enable / Disable banner Status.</li>
<li>* Click 'Banner Manager' under 'Banner Ad Slideshow' to delete banner.</li>
<li>* Click 'Banner Manager' under 'Banner Ad Slideshow' to edit banner description, URL etc.</li>
<li>* No cron job require to automate banner expiring date</li>
</ul>
</p>

== Installation ==
<p>
<ul>
<li>1. Upload files of Banner Ad Slideshow to the `/wp-content/plugins/`</li> 
<li>2. Activate the plugin through the 'Plugins' menu in WordPress</li>
<li>3. At backend or admin page, put ShortCode /*[banner_ad_slideshow]*/ to add Banner Ad Slideshow to the page. Please always remember 
to remove [/* */] from the Shortcode.</li>
<li>4. At frontend, put ShortCode /*if ( shortcode_exists( 'banner_ad_slideshow' ) ) { echo do_shortcode('[banner_ad_slideshow]');} */ to 
add Banner Ad Slideshow to the page. Please always remember to remove [/* */] from the Shortcode.</li>
<li>5. At backend , click 'Help' under 'Banner Ad Slideshow' menu in WordPress for help user guide.</li>
</ul>
</p>
